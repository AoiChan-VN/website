# Aoi Interface

A cinematic static website architecture designed for:

- GitHub Pages
- Vanilla JavaScript
- Modular scaling
- IOS inspired interface
- Anime fantasy atmosphere

## Features

- Data-driven content
- Dynamic markdown rendering
- Overlay interaction panel
- Low-end mobile optimization
- PWA support
- Modular CSS architecture

## System

Every content source is mounted through:

`data/aoi-file.json`

No core rebuild required when adding new content collections.
